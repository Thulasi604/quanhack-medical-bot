# Medical Lab Telegram Bot

Telegram bot for booking lab tests.

## How to use
1. Send `/start`
2. Send `/book`  
3. Type: `blood test, tomorrow, 10am`
4. Type: `CONFIRM`

## Example
- User: /book
- Bot: Please send test type, date, time
- User: blood test, tomorrow, 10am
- Bot: Reply CONFIRM to book
- User: CONFIRM
- Bot: ✅ Booking confirmed!
